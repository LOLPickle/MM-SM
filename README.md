MM&SM — це легкий, сучасний та повністю безкоштовний додаток для створення звуку у відеоіграх. Поєднує в собі потужний генератор звукових ефектів (як у Bfxr) та простий музичний синтезатор з базовими інструментами. Ідеальне рішення для інді-розробників, геймдизайнерів та всіх, хто хоче швидко створити унікальний звуковий супровід.

---

✨ Ключові можливості

· 🎛️ Генератор звукових ефектів — створюйте класичні ігрові звуки: вибухи, стрибки, збирання предметів, постріли та багато іншого.
· 🎹 Вбудований синтезатор — прості інструменти для створення мелодій: синтезатор, піаніно, гітара.
· 🎨 Сучасний інтерфейс — чистий та інтуїтивно зрозумілий дизайн на основі CustomTkinter.
· ⚙️ Гнучке налаштування — змінюйте вигляд редактора під свої потреби.
· 🚀 Легкий та швидкий — миттєвий запуск та мінімальне споживання ресурсів.

🚀 Встановлення та запуск

Варіант 1: Запуск .exe (рекомендовано)

Найпростіший спосіб почати роботу:

1. Перейдіть у розділ Releases цього репозиторію.
2. Завантажте останню версію файлу MM&SM.exe.
3. Запустіть файл. За бажанням, ви можете створити ярлик на робочому столі для швидкого доступу.

Примітка: Антивірус може реагувати на .exe файли, створені з Python. Це звичайна ситуація. Ви можете додати файл до винятків або скористатися варіантом 2.

Варіант 2: Запуск з вихідного коду (для розробників)

Якщо ви бажаєте запустити додаток з вихідного коду:

1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/your_username/MM&SM.git
   cd MM&SM
   ```
2. Встановіть необхідні залежності:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустіть додаток:
   ```bash
   python main.py
   ```
